Погодный виджет

Описание функционала

GET ALL /api/weather

Получить погоду во всех городах.

GET CITY /api/weather/:city

Получить прогноз погоды для конкретного города (например, Москва, Токио и т.п.).

POST /api/weather

Добавить новый город и его погоду.

PUT /api/weather/:city

Обновить данные о погоде для города.

DELETE /api/weather/:city

Удалить город из базы данных.
