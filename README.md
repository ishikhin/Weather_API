# Weather API

Express приложение - погодный виджет с фиктивными данными.

## Функциональность

CRUD операции для городов и погодных данных
Полноценное API с поддержкой всех HTTP методов (GET, POST, PUT, DELETE)
Фильтрация погоды по городу через параметры маршрута
Логирование всех запросов через кастомный middleware
RESTful API
Файловое хранилище данных (JSON-файл)

## Установка и запуск

1. Клонировать репозиторий:
```bash
git clone https://github.com/ishikhin/Weather_API
cd event-feed-api
```

2. Установить зависимости:
```bash
npm install
```

3. Запустить сервер:
```bash
npm start
```
